# Guwahati Urban Flood Analysis 🌊
**AEC Social Internship | 3rd Semester (NEP 2020 Batch)**

## 📌 Project Overview
This project is a socio-technical investigation into the recurring urban flooding issues in Guwahati, Assam. Conducted as part of the mandatory Social Internship at **Assam Engineering College**, the study focuses on identifying drainage bottlenecks and the socio-economic impact on local communities.

## 📂 Key Deliverables
To ensure transparency and academic documentation, the following resources are available in this repository:

* 📄 [**Final Technical Report**](./AEC_Social_Internship_Report.pdf) - A detailed analysis of field observations and mitigation strategies.
* 📊 [**Presentation Slides**](./AEC_Social_Internship_Presentation.pdf) - Summarized findings presented to the AEC faculty panel.

## 🔍 Focus Areas
- **Drainage Analysis:** Identification of infrastructure gaps in urban wards near Jalukbari.
- **Community Impact:** Survey-based data on commute disruptions and business losses.
- **Proposed Solutions:** Suggestions for technology-integrated drainage monitoring and waste management.

## 🛠️ Tools Used
- **Field Survey:** Physical inspection and community interviews.
- **Data Documentation:** Microsoft Word.
- **Presentation:** PowerPoint.
- **Version Control:** Git & GitHub.
